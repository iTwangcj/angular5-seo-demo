USER MANUAL

创建文档

登录/注册
点击页面顶部“注册”进入注册页面，设置FEMock ID、登陆密码即可完成注册。
创建团队/组

进入“团队"页面：

1、点击“创建团队”，设置团队名称、权限（仅团队成员可见：只有加入在该团队中的成员可见，全员可见：系统中所有用户都可见）、描述即可完成创建团队。

2、创建团队成功后，点击“创建小组”设置小组名称、描述、添加小组成员即可完成创建小组。

创建项目
进入“项目”页面，点击“✚”，设置项目名称、归属、描述即可完成创建项目。
创建模块/接口

1、点击“某项目”进入项目主页，然后点击“创建模块”，设置名称、描述即可完成创建模块。

2、创建模块成功后，点击“创建接口”，设置接口名称、请求方法、请求地址、描述即可完成创建接口。

注意：完成创建接口并未保存，当编辑完后点击"保存"按钮才被提交。

工作区

工作区状态
选择已创建好的“某接口"，点击“编辑”，此时已激活编辑状态。点击“取消编辑"即锁住工作区，不可编辑内容。

导出接口数据
在文档编辑区，点击右上角“•••” => 导出JSON、导出PDF、导出Markdown即可导出你想要的格式。
接口测试
在文档编辑区，点击右上角“运行”，查看接口状态及返回结果。
